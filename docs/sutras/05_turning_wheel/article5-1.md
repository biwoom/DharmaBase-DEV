# 초전법륜 (The First Turning of the Wheel)

## 1. 녹야원에서의 재회

깨달음을 얻으신 부처님께서는 갠지스 강을 건너 바라나시의 녹야원(사슴 동산)으로 향하셨습니다. 그곳에는 고행을 함께했던 다섯 수행자(오비구)가 머물고 있었습니다. 그들은 처음에 부처님을 외면하려 했으나, 부처님의 위엄 있는 모습에 저절로 일어나 자리를 권했습니다.

## 2. 중도(Middle Way)의 가르침

부처님께서는 먼저 **양 극단에 치우치지 않는 길**, 즉 중도에 대해 설하셨습니다.

* **쾌락의 추구**: 감각적 욕망에 빠지는 것은 저급하고 무익하다.
* **극단적 고행**: 육체를 괴롭히는 것 또한 고통스럽고 무익하다.

## 3. 사성제 (Four Noble Truths)

중도를 실천하는 구체적인 방법으로 네 가지 성스러운 진리를 설하셨습니다.

1. **고제 (苦諦)**: 삶은 근본적으로 괴로움이다. (생로병사)
2. **집제 (集諦)**: 괴로움의 원인은 집착과 갈망이다.
3. **멸제 (滅諦)**: 집착을 없애면 괴로움이 사라진 상태(열반)에 이른다.
4. **도제 (道諦)**: 열반에 이르는 길은 **팔정도(Eightfold Path)**이다.
